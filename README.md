## Overview

This is the backend service for the Expense Management System. It provides secure RESTful APIs for user authentication, expense management, and data storage. The backend handles business logic, database operations, and API security.

---

## Features

- User authentication (Register / Login)
- JWT-based authorization
- Create, read, update, and delete expenses (CRUD)
- Expense categorization
- Secure password hashing
- Role-based access (optional)
- Environment-based configuration

---

## Tech Stack

- Node.js
- Express.js
- RESTful API architecture
- MongoDB / MySQL (based on configuration)
- JWT for authentication
- bcrypt for password encryption
- dotenv for environment variables
